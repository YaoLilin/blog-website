
# 需求说明
需要制作一个博客网站，可用来展示和编写博客。  
前端请写在 frontend 文件夹里，后端写在 server 文件夹里

## 网站架构
前端：使用 React 框架，构建工具使用 vide，使用 shadcn/ui 组件
后端：使用 SpringBoot 框架，持久层框架使用 spring data，数据库使用 mysql ，缓存使用 redis
其它：使用 nginx 反向代理后端接口

## 网站风格
风格以简约为主，可参照 https://ui.shadcn.com/ 网站的风格

## 详细说明
### 前台页面
可参考 https://ui.shadcn.com/ 网站的布局  
#### 导航栏
最上方显示导航栏，导航栏的左边显示“姚礼林的博客”，点击文字可跳转到首页，接着是导航。  
导航按钮：
- 第一个导航按钮是“文章”，点击文章时跳转到文章分类页面，鼠标移动到此按钮时下方展开文章类型的分类，鼠标移动到分类时在右边展开子分类，鼠标移动到子分类时再展开其子分类，以此类推，点击分类时可跳转到对应分类的页面
- 第二个按钮是“关于我”，点击后跳转到关于我的页面
导航的右边是搜索框，可搜索文章内容  
再右边是深色和浅色主题切换按钮  
再右边是github按钮，显示一个github图标即可，点击后可跳转到github主页  
#### 欢迎页面
此页面作为访问此网站的第一个页面，页面上方显示我的座右铭，为：“我们得为人类做点什么”  
##### 推荐分类
接下来是显示推荐的文章分类，可自定义推荐的文章分类，每个文章分类使用一个带圆角的矩形图形包裹，横向排列
##### 推荐文章
接下来是推荐文章，推荐文章可最多显示10条，列表显示
##### 最近文章
接下来是最近文章，列出最近创建的文章，列表显示，最多显示20条

#### 文章分类页面
文章分类有两种显示形式
##### 一、图标显示
首先展示所有的一级文章分类，点击分类后再展示二级文章分类（一级文章分类消失），以此类推，每个文章分类显示中，能显示该分类下的文章，分类和文章在页面中是不同的板块，分类板块显示在上面，文章板块显示在下面，文章使用列表显示，每行一个文章。  
分类使用大的圆角矩形包裹，矩形内可显示分类定义的封面  
在分类板块上方，显示当前分类的路径，比如：分类一/子分类一/当前分类，点击路径中的分类，可跳转到指定分类

##### 二、树形显示
树形显示分类，首先只显示一级分类，点击可展开下级分类，以此类推，树形结构中可显示文章，文章和分类要用图标区分，分类图标为文件夹图标，点击文章可跳转到该文章页面

#### 文章页面
布局为：左目录，中间为文章内容，右边为文章的下级标题  
##### 目录
目录上方显示分类路径，比如：分类一/子分类一/当前分类，点击路径中的分类可在目录中显示该分类的下级分类以及文章，点击分类可显示对应分类的下级分类和文章，并修改目录上面的分类路径，点击目录中的文章后目录变为文章的目录。  
文章目录只显示文章的二、三级标题，其它标题放到右边的目录显示，点击标题可跳到文章对应位置
##### 文章内容
在文章上面显示头部日期，包括创建日期，最后修改日期，阅读数，其中阅读数只有登录了系统管理员才可以查看
文章用 markdown 文档显示
##### 下级标题目录
文章右边显示当前三级标题的下级标题，如果没有就不显示，点击标题可跳到文章对应位置  
##### 编辑功能
文章默认是预览模式，可通过按钮切换到编辑模式，只有登录到了系统管理员才显示编辑按钮，在编辑模式下提供源码编辑和实时预览编辑功能，
实时预览编辑可在同一个页面，既能编辑md文档也能预览md文档，点击保存按钮后保存文章。
##### 阅读数统计
不统计系统管理员的阅读，同一个浏览器内重复访问，只算1次阅读数

### 后台页面
#### 登录
在导航栏上留一个按钮，点击可弹出登录对话框，不需要输入账号，只需要输入管理员密码，登录后登录状态保持7天，7天过后登录过期。登录密码通过服务器上的配置文件进行配置。
#### 后台页面
登录后在导航栏的最右边显示一个按钮，点击后可跳转到后台管理页面
#### 导航栏
页面左侧为导航栏菜单，菜单可为树形结构，可分为一级菜单、二级菜单。
#### 文章管理
文章管理可管理所有文章。  
##### 分类树
左侧为文章分类目录，树形结构，可添加目录和修改目录名称（通过放入服务器内的文档不能修改目录名称），可通过拖拽移动目录，如果是上传到服务器的文档，移动目录时也同步移动服务器内的目录。
##### 文章列表
选择分类后，右边出现该目录的文章，以列表形式显示，并显示文章的创建日期和阅读数，支持搜索，点击文章后在当前页面显示文章的预览，就和前台的文章内容一样，点击编辑按钮切换到编辑，点击保存后进行保存。 可通过返回按钮返回到列表。  
可通过创建按钮添加文章。  
所有文章都只能为 markdown 格式
##### GIT 同步
- git提交：自动检测上传到服务器中的文件是否有git，如果有git可通过后台管理进行git提交，推送到远程仓库
- 远程仓库管理：可添加git远程仓库
- git 拉取：可从远程仓库拉取，更新到服务器，如果服务器还没有仓库的文件，则在指定目录下进行创建
自动检测
#### 统计页面
- 显示总阅读数，统计所有文章的阅读数
- 显示总阅读数增长趋势图表
- 显示文章阅读排行，从高到低显示10条阅读量最高文章，每个文章后面都显示阅读数
#### 设置
对于博客网站的一些设置
- 针对服务器管理的文件：附件存放位置配置：
    - 在根目录
    - 在当前文件夹
    - 在当前文件夹指定的文件夹
    - 指定路径
- 座右铭：可配置在首页显示的座右铭

### 将文档上传到服务器
可通过将文档上传到服务器内的指定目录，来添加文章，上传之后，目录内的所有文件夹名变为分类，可在博客上显示里面的所有文章。
